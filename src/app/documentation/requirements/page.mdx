### Consideraciones generales
- WPA friendly
- tener diferente ruta para usuarios logueados y deslogueados (home?)
- tener un blog y una manera de publicar eventos y calendario
- tener una forma de mostrar y vender productos
- marquesina de información que se pueda prender y apagar
- fall tree analysis / tech support


### Features por Rol de usuario:
#### 1. rol de alumno
- **sign up** 
    - email
    - y el código de los gimansios a los que pertenece
- **login / logout / password change**
- **anotarse y desanotarse a clases que ya existan**
- **ver y editar el perfl propio**
    - datos personales
    - histórico de clases en las que estuvo
    - clases futuras
    - rutina 
    - datos de salud?
    - configuracion notificaciones y newsletter

#### 2. rol de profesor
    - **sign up / log in / log out / password change**
    - **ver y editar el perfl propio**
    - **gestionar clases asignadas a ese profesor**
        - anotar y desanotar alumnos a las clases
        - ver quienes van a ir a sus clases 

#### 3. Rol de Administrador
    - **sign up / login / logout / password change**
    - **ver y editar el perfl propio**
    - **gestionar usuarios**
        - Altas y bajas de usuarios / confirmación de cuenta
        - Ver toda la información de perfil de todos los usuarios
            - editar las clases a las que está anotado el usuario
            - editar las rutinas del usuario
            - editar paquetes la clases
        - gestionar notificaciones

    - **gestionar profesores**
        - asignar rol de profesor a un usuario normal
        - asignar clases a un profesor
        - ver perfil y clases asignadas a ese profesor
        - gestionar notificaciones

    - **gestionar clases**
        - Crear clases
        - editar las clases
        - poner y sacar profesores
        - poner y sacar alumnos
        - ver todo el calendario de clases

    - **Vistas de data del histórico**
